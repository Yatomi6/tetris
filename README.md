# Tetris (C + raylib)

Ce projet est un projet d'apprentissage.
Je le développe pour améliorer mes compétences en C.

## Objectif

Recréer le plus fidèlement possible le jeu Tetris

## Etat actuel

Le jeu n'est pas terminé.
Le code évolue régulièrement.

## A faire

Fonctionnalites encore à implémenter:
- save/hold de pièces
- prévisualisation des pièces suivantes (next queue)
- high score

## Lancer le projet

Prerequis:
- `gcc`
- `make`
- `raylib`

Commandes:
- `make`
- `make run`
